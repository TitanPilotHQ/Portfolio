## Team composition (Phase A — today)

> "Titan Pilot is currently founder-led and intentionally operated by a
> single founder."
*(Source: `INTAKE-REQUEST.md` §8.1, Emad's direct answer, 2026-07-12, quoted
verbatim.)*

This is the same underlying team-composition fact canonically owned by
Module 01 (`01-brand-bible/01-identity.md`, "Company stage") — see that
file for its sourcing. Intake §8.1 is this module's own direct grounding
source; it is not a second, independent derivation of the fact.

Plainly, as of this writing:

- No job postings exist.
- No hiring process exists.
- No compensation or equity framework exists.

None of these are gaps in the sense of something missing that should exist
already — see the next section.

## "Intentionally" is load-bearing

The verbatim quote above says "intentionally operated by a single founder,"
not merely "operated by a single founder." This file reads that word as
signaling deliberateness rather than constraint — that reading is this
file's own inference from the word choice, not a separate statement from
Emad. `02-anticipated-hiring.md` documents the real, sourced conditions
under which the single-founder state changes.

## What this file deliberately does not invent

- A team size beyond "single founder" — no headcount number, no informal
  team-size estimate, no characterization like "small team" beyond the
  "intentionally operated by a single founder" language already quoted.
- Any informal contractor, freelancer, or advisor arrangement — none is
  sourced anywhere in this program. This file does not assume Titan Pilot
  has no such arrangements either; it simply does not state one, because
  intake §8.1 does not mention one.
- A hiring timeline for Phase A itself. There is none to invent: Phase A, by
  definition, is the phase in which no hiring occurs. A timeline belongs to
  Phase B, and is not invented there either — see
  `02-anticipated-hiring.md`'s own "what this file deliberately does not
  invent" section.

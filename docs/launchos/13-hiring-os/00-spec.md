# Module 13 — Hiring OS — Spec

**Status:** Ready to build
**Depends on:** Module 01 (principles, company stage — this module cites,
does not restate, the founder-led team fact), Module 10 (legal-entity
Phase A/B — this module cites, does not restate, for the employment/
contracting dependency), Module 11 (finance Phase A/B pattern — this module
follows the same discipline against inventing compensation figures)
**Purpose:** The canonical home for Titan Pilot's hiring reality — currently
none — and the Phase A/B structure for the anticipated first-hire roles and
process, without inventing compensation, equity terms, or a timeline that
isn't sourced.

## Sources

Grounded in `INTAKE-REQUEST.md` §8 (Emad's direct answers, 2026-07-12).

## Global Constraint 1 application

Intake §8.2 explicitly states "No compensation or equity policy has been
finalized." This module must not produce a salary range, an equity band, a
title/level structure, or a specific hiring timeline/date — none is
sourced. It documents the real anticipated role categories and their stated
trigger (successful design-partner validation) without inventing the
specifics of what a real offer would look like.

## Global Constraint 3 application

- **Phase A (today):** founder-led, intentionally operated by a single
  founder (intake §8.1, verbatim). No open roles, no hiring process, no
  compensation/equity framework exists.
- **Phase B (future):** initial hires expected after successful
  design-partner validation (intake §8.2), in engineering, customer
  success, product design, and sales — framed conditionally, no date, no
  invented comp/equity structure, no invented headcount number, no invented
  org chart.

## Global Constraint 4 application

- Anticipated role categories and their trigger are new to this module —
  no other module has stated them, so this module is the canonical owner.
- "Founder-led, single founder" team-composition fact is canonically owned
  by Module 01 (`01-brand-bible/01-identity.md`, "Company stage") — this
  module cites it rather than restating its sourcing.
- Legal-entity status (relevant since formal employment typically requires
  an incorporated entity as employer) is cited from Module 10/07, not
  restated.
- Finance-adjacent facts (no comp/equity budget stated) stay in this module
  since they're hiring-specific, not general financial-operations facts
  already owned by Module 11 — but this module follows Module 11's
  established discipline against inventing any figure.

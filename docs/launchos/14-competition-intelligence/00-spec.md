# Module 14 — Competition Intelligence — Spec

**Status:** Ready to build
**Depends on:** Module 04 (competitive positioning map — this module cites,
does not restate, the 8 real named competitors and their cluster/
differentiator analysis)
**Purpose:** The canonical home for Titan Pilot's competitive monitoring
*scope* — what categories to track, and the standing repository this module
establishes for competitive research going forward — as distinct from
Module 04's one-time positioning analysis of the 8 already-named
competitors. No monitoring cadence, methodology, or tooling exists yet;
this module documents the scope honestly rather than inventing a process
that isn't sourced (see `02-research-process.md`).

answers, 2026-07-12).

## Global Constraint 4 application — the central boundary of this module

Module 04 (`04-positioning/02-competitive-positioning-map.md`) is the
canonical owner of the competitive positioning map: the 8 real named
competitors, their clusters, and their differentiators. This module does
NOT restate that list, does NOT re-derive cluster assignments, and does NOT
re-describe any individual competitor's `theyAre`/`difference`/`concede`
fields. Where this module needs to reference a named competitor (e.g., to
note it as an example of a category already covered), it cites Module 04.

This module's actual canonical subject — new to LaunchOS, owned nowhere
else — is the monitoring *process*: what categories intake §9.1 says to
expand tracking to, the fact that no structured competitive research
repository existed before this module (intake §9.2), and this module's role
as that repository going forward.

# Module 03 — Messaging Bible — Spec

**Status:** Ready to build (no blocked facts)
**Depends on:** Module 01 (Brand Bible — voice, principles, never-claims) and
Module 02 (Story Architecture — narrative arc, scenes)
**Purpose:** Where Module 01 sets rules and Module 02 sets structure, this
module sets the actual **words** — the message hierarchy (what's primary vs.
supporting), a terminology glossary (which words are approved, which need a
mandatory companion phrase, which are banned), and channel-specific message
variants assembled only from already-approved building blocks.

## Relationship to Module 04 (Positioning)

Messaging Bible answers "what do we say and how do we say it." Positioning
(Module 04) answers "where do we sit relative to alternatives, and for whom."
They share the same source material (the competitor comparison, the category
definition) but Messaging Bible is about word-level execution across
channels; Positioning is about the market map. Build Module 04 next, citing
this module for word choice rather than re-deriving it.

## Sources

Same grounding rule as prior modules. This module does not invent new
claims — every channel message is either (a) a direct quote from live copy,
or (b) an assembly of already-approved fragments from Module 01's elevator
pitches or Module 02's narrative scenes, with no new factual content
introduced at the assembly step.

## Structure

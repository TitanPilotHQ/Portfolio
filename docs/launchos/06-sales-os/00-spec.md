# Module 06 — Sales OS — Spec

**Status:** Ready to build. The ICP sub-file (`00-icp.md`) was already built
early, out of numeric order, because Modules 03 and 04 needed the canonical
ICP fact before this module's own turn (Global Constraint 4). This spec
covers the rest of the module: the actual sales process, tooling reality,
and how sales conversations should draw on other modules' content rather
than re-deriving it.
**Depends on:** Module 01 (Brand Bible — voice), Module 02 (Story
Architecture — the Decision Walkthrough scene and the Technical/partner
audience variant used in Stage 5), Module 03 (Messaging Bible — channel
messages, terminology), Module 04 (Positioning — formal positioning
statement, competitive map), Module 05 (Pricing System — what can and cannot
be said about pricing in a sales conversation).
**Purpose:** The canonical home for how Titan Pilot actually sells today —
process, tooling, and the rule that sales conversations assemble from
already-approved content rather than improvising new claims in the room.

## Sources

Grounded in `INTAKE-REQUEST.md` §2 (Emad's direct answers, 2026-07-12) — an
internal planning input describing the real, current, manual process. Per
Global Constraint 1, this module documents the process as it actually is
today; it does not invent a more mature sales process (a formal pipeline
tool, named sales stages with conversion metrics, a quota system) that
doesn't exist yet.

## Global Constraint 3 application

Sales process touches how the company engages prospective customers, which
is adjacent to contracting once a design-partner relationship becomes
commercial. Applied narrowly here: the process itself (discovery →
demo → technical discussion → design-partner evaluation → onboarding) is
already real and described as Phase A fact; nothing in this module invents a
Phase B sales process, since intake §2 doesn't describe one — Module 06
does not speculate about what sales will look like post-incorporation beyond
what's already stated elsewhere (e.g., Module 05's pricing Phase B).

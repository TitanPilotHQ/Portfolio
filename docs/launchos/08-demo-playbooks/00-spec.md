# Module 08 — Demo Playbooks — Spec

**Status:** Ready to build
**Depends on:** Module 01 (elevator pitches), Module 02 (Mental Model and
Decision Walkthrough scenes), Module 03 (terminology), Module 04
(competitive map), Module 06 (Stage 4 of the sales process, ICP)
**Purpose:** The canonical home for how a live product demonstration
actually runs. Per Global Constraint 4, this module does not redefine the
Mental Model, the Decision Walkthrough, elevator pitches, or the ICP — it
sequences and times them into an actual demo script, which is the new,
Module-08-owned content: the specific order, transitions, and pacing of a
live demo, not the underlying facts the demo is built from.

## Sources

Grounded in Module 02's existing narrative scenes and Module 06's sales
process (specifically Stage 4, "product demonstration," described in intake
§2.4 as "tailored," not scripted-identically-every-time). No new factual
claims about the product are introduced here — every claim in this module
traces to a citation in an earlier module.

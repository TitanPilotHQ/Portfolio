# Asset Library — 01. LaunchOS Catalog

An index of every file LaunchOS v1 has produced, grouped by module. This
file is a table of contents, not a summary — it names what exists and where,
and does not restate any fact, figure, or quote from any module's content.
To find an actual claim (a price, a competitor name, a pipeline status,
anything), open the cited module's own file. Do not treat any description
below as a substitute for reading the source file.

Grounded directly in the verified `docs/launchos/` directory listing
supplied for this module's build (Module 16 spec, Sources) — not
independently re-derived or guessed.

## Program-level files

Not part of any numbered module; scope the program as a whole.

- `00-launchos-master-plan.md` — global constraints, module readiness
  matrix, process, sequencing rationale.
- `INTAKE-REQUEST.md` — the batched factual intake Emad answered on
  2026-07-12, grounding the modules that needed a founder-supplied fact.
- `PROGRESS.md` — append-only closure ledger, one entry per module
  closure or program milestone.
